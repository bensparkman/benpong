# BEN PONG

A self-contained browser Pong game. The ball is a real head with alpha-accurate
silhouette physics, spin, bulging/direction-tracking eyes, two levels (Ben → Ally),
a 1–10 difficulty picker, procedural funk + demonic-cat-EDM soundtracks, themed
"snipe" taunts, and randomized fart SFX. Everything is embedded in `index.html`
(no external assets). Served via GitHub Pages.
